# Product Overview

Simsoft Twig is a PHP wrapper library around the Twig 3.x template engine. It
provides a simplified, configuration-driven API for setting up and rendering
Twig templates.

## Core Capabilities

- Configuration-based initialization (paths, caching, debug, charset, timezone)
- Template rendering and display (full templates and individual blocks)
- Filesystem loader with namespace support
- Simplified extension authoring via a base `Extension` class
- Fluent API for adding filters, functions, and global context at runtime

## Target Audience

PHP developers who want a streamlined interface for integrating Twig into
applications without manually wiring up the loader, environment, and extensions
each time.
